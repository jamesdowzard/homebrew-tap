# Homebrew Tap

Personal Homebrew tap for my projects.

## Installation

```bash
brew tap jamesdowzard/tap
```

## Available Casks

### Rodin

Local voice dictation powered by Whisper - an open source alternative to Wispr Flow.

```bash
brew install --cask jamesdowzard/tap/rodin
```

Features:
- 100% offline transcription using faster-whisper
- Multiple recording modes (hold, toggle, double-tap)
- Optional AI text cleanup (Ollama, OpenAI, Anthropic)
- Stats tracking for your dictation history
- Native macOS app with menu bar integration
